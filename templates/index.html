💡 Project Overview

This AI system uses EfficientNetB3, a deep convolutional neural network, to classify brain MRI scans into four categories, aiding early tumour detection with high accuracy.

Trained on a dataset of MRI images, the model leverages data augmentation and transfer learning to achieve robust performance, as shown in the training accuracy and loss plots.

📌 Tumour Types Detected:

  • 🧠 Glioma — Tumour in glial cells of the brain.
  • 🧠 Meningioma — Tumour from the brain’s meninges.
  • ✅ No Tumour — Healthy brain MRI.
  • 🧠 Pituitary — Tumour near the pituitary gland.

✨ Key Features:

  • 📸 Real-time MRI image upload
  • 📈 AI predictions using EfficientNetB3
  • 💾 Secure image processing
  • 📊 Displays prediction with confidence scores
  • 🎯 Supports early tumour detection

🛠️ Algorithm Details

The system is powered by a deep learning model built with TensorFlow and Keras, utilizing the EfficientNetB3 architecture pre-trained on ImageNet.

  • 🔬 Model Architecture: EfficientNetB3 with global average pooling, dropout layers, and a softmax output for 4 classes.
  • 📉 Training: 30 epochs with Adam optimizer, categorical crossentropy loss, and data augmentation (rotation, zoom, flip).
  • ⚙️ Callbacks: EarlyStopping, ReduceLROnPlateau, and ModelCheckpoint to optimize training.
  • 📊 Performance: Evaluated on test data with accuracy metrics plotted for training and validation.

The model is saved to Google Drive for deployment, ensuring scalability and accessibility.
